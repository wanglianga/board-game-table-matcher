# 桌游吧 · 组局拼桌

## 原始需求

> 请制作桌游吧组局拼桌页面，Vue3 页面呈现游戏库、玩家人数、预计时长、规则复杂度、主持人、桌号、拼桌进度、教学需求和饮品套餐。玩家在手机上筛选推理、策略、派对或合作类游戏，看到已有成员水平、缺几人、开局倒计时、能否中途退出和主持人位置；店内屏幕滚动展示等待桌、已成局、临时换桌和即将讲规则的场次。页面可以热闹，但不能让游戏封面抢掉关键信息，玩家一眼要知道自己适不适合加入、还有多久开、到哪张桌集合。

## 项目简介

桌游吧组局拼桌平台，包含两个视图：

- **玩家手机端** (`/`)：分类筛选、对局卡片、拼桌详情、加入/退出、饮品套餐
- **店内大屏端** (`/screen`)：等待桌滚动、已成局展示、即将讲解高亮、换桌通知

## 技术栈

- Vue 3 + TypeScript + Vite
- Tailwind CSS 3
- Pinia 状态管理
- Vue Router 4
- Lucide Vue Next 图标

## 启动方式

### 前置要求

- Node.js 18+
- npm

### Docker 一键启动（推荐）

#### 前置要求

- Docker
- Docker Compose

#### 启动步骤

```bash
docker compose up --build
```

访问地址：http://localhost:8080

后台运行：

```bash
docker compose up --build -d
```

停止服务：

```bash
docker compose down
```

### 本地开发启动

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动服务

```bash
npm run dev
```

访问地址：http://localhost:5173

### 页面路由

| 路由 | 说明 |
|------|------|
| `/` | 玩家手机端 |
| `/screen` | 店内大屏端 |
